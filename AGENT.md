# RAG Agent Helper

An autonomous agent that assists with RAG Agent development.

## Capabilities

This agent can:
- Write and modify code in Python and TypeScript
- Run tests and analyze results
- Execute development commands
- Search for solutions to common problems
- Work independently on assigned tasks

## Usage

Invoke this agent when you need help with:
- Code refactoring
- Bug fixing
- Feature implementation
- Test writing
- Documentation updates

## Agent Behavior

The agent is designed to be self-directed and will:
1. Analyze the task requirements
2. Explore the codebase as needed
3. Implement solutions following project patterns
4. Test the implementation
5. Report results and seek feedback when needed
